# 天选之饭 —— 智能餐饮推荐小程序（项目说明文档）

## 一、项目概述

“天选之饭”是一款基于微信小程序开发的智能餐饮推荐平台，旨在解决用户“吃什么”的决策困难问题。系统结合随机推荐机制、地理位置服务与用户社区互动，构建一个集“推荐 + 分享 + 个性化选择”为一体的餐饮决策平台。

项目通过小程序作为前端入口，结合位置服务与数据管理，实现从“随机选择”到“精准推荐”的用户体验升级。

---

## 二、核心功能模块

### （1）随机推荐（核心功能）

首页“抽一抽”功能：

- 随机推荐餐厅或菜品
- 提供趣味化选择体验
- 降低用户决策成本

👉 解决问题：不知道吃什么

---

### （2）餐厅浏览与详情

#### 餐厅列表页面（restaurant）

- 展示餐厅信息
- 支持分类浏览

#### 餐厅详情页面（restaurantDetail）

- 展示餐厅详细信息
- 包含菜品列表

---

### （3）菜品详情展示

#### foodDetail 页面

- 展示具体菜品信息
- 支持用户查看详细介绍

---

### （4）附近推荐（基于位置）

#### nearby 页面

基于用户位置实现：

- 附近餐厅推荐
- 提供地理位置服务
- 提升推荐相关性
